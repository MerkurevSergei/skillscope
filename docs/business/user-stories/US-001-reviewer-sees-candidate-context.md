# US-001 Reviewer Sees Candidate Context

Status: draft

## Story

As a reviewer, I want to see candidate information before preparing an interview, so that questions can match the candidate's target role.

## Acceptance Criteria

- Reviewer can identify the candidate by full name.
- Reviewer can see candidate target role when present.
- Reviewer can proceed to interview preparation from candidate context.
