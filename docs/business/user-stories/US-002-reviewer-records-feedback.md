# US-002 Reviewer Records Feedback

Status: draft

## Story

As a reviewer, I want to record a score and summary after an interview, so that the interview result is available later.

## Acceptance Criteria

- Reviewer can set score from 1 to 5.
- Reviewer can add optional summary.
- Saved feedback remains linked to the interview.
