# Business Overview

Status: draft

## Product Summary

SkillScope — платформа для подготовки и проведения интервью, где можно хранить кандидатов, интервью, вопросы и обратную связь ревьюеров.

## Actors

### Candidate

Человек, для которого проводится интервью.

### Reviewer

Пользователь системы, который оценивает интервью и оставляет обратную связь.

### System

Backend и frontend SkillScope, которые хранят данные, показывают сценарии работы и обеспечивают доступ к API.

## Core Concepts

### Candidate

Карточка кандидата содержит имя, email и целевую роль.

### Interview

Интервью привязано к кандидату, имеет название, статус, дату проведения и связанные вопросы.

### Interview Question

Вопрос относится к интервью, содержит текст вопроса, ожидаемый уровень и порядок сортировки.

### Question Card

Переиспользуемая карточка вопроса содержит текст вопроса, классификационные tags, подсказки для собеседующего и критерии оценки ответа.

### Interview Feedback

Обратная связь относится к интервью. Может быть связана с ревьюером, содержит оценку и текстовое резюме.

## Current Functional Scope

- Просмотр health endpoint backend: `GET /api/health`.
- Хранение базовой доменной модели в PostgreSQL.
- Подготовленная структура frontend/backend для дальнейшей разработки.

## Out Of Scope For Now

- Полная модель аутентификации и ролей.
- CRUD API для доменных сущностей.
- Workflow статусов интервью.
- Полнотекстовый поиск, аналитика и отчеты.
